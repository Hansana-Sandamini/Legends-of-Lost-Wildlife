# Legends of Lost Wildlife

A digital archive documenting extinct animal species with detailed information, images, and conservation resources.
<br />

## Features

- 🦕 Comprehensive extinct animal database
- 🐘 Categorized by species types (Mammals, Fishes, Breeds, Insects, Reptiles and Invertebrates.)
- 📚 Educational content about extinction causes
- 🔍 Search functionality
- 📱 Fully responsive design
- 🎥 Embedded educational videos
<br />
